# Impact Typer — 💥 爆音・爆破エディタ

> **普通のタイピングをハリウッド映画に変える**、究極にバカバカしい爆音テキストエディタ。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/impact-typer)

---

## 🎬 これは何？

「ただ *了解です* と2文字打っただけなのに、**世界を救うミサイルのスイッチを押したかのような大迫力**」を味わえるエディタ。

1文字タイプするたびに：
- 💣 **爆音**が鳴り響く（Web Audio APIで完全生成、外部ファイル不要）
- 📳 **画面全体が激しくシェイク**する
- ✨ **火花・紙吹雪・衝撃波**が画面を埋め尽くす
- 🌡️ **連続タイピングでエフェクトがどんどん激化**するコンボシステム

## 🎮 機能

| 機能 | 説明 |
|------|------|
| **4種の爆音モード** | 💣 爆発 / ⚡ レーザー / ⌨️ メカキー / ☢️ 核爆発 |
| **コンボシステム** | 連続入力でエフェクト激化。5/10/20/30/50/100コンボで特別演出 |
| **強度メーター** | タイピング速度に応じてINFERNO〜APOCALYPSEまで変化 |
| **実績システム** | 解除するたびにトースト通知 |
| **コナミコマンド** | ↑↑↓↓←→←→BA でGODMODE発動 |
